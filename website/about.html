Type Patterns is a pattern generator in which every letter is assigned a number and is combined with the other letters in each word to create a color. Each word is one unit of the pattern. The length of each unit is determined by the length of each word. The height of all the units is determined by the average length of each word. Punctuation and spaces are white, providing white space to the repeated pattern.

Experiment with different combinations of letters, words, sentences, and even full paragraphs to produce a unique pattern.

Type Patterns was created by Jessica Metro using Processing and Processing.js. Check it out on GitHub.
